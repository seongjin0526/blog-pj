# Tag Discovery UI (Codex)

## 목적
네비바에서 태그를 빠르게 탐색할 수 있도록 `상위 태그`, `더보기`, `검색`, `자동완성`을 일관된 방식으로 구현한다.

## 입력 요구사항
- 태그는 빈도 내림차순으로 집계한다.
- 상위 태그만 1줄 표시하고 나머지는 `더보기`로 토글한다.
- `검색` 버튼으로 검색 입력창을 열고, 입력 중 유사 태그를 최대 N개 추천한다.
- 엔터/버튼/추천 클릭 시 `?tag=<name>` 필터 페이지로 이동한다.
- 향후 제목/본문 검색 확장을 위해 검색 매칭 함수를 분리한다.

## 구현 체크리스트
1. 컨텍스트 데이터
- `navbar_top_tags`, `navbar_all_tag_items`, `navbar_current_tag` 제공

2. 템플릿
- `base.html` 네비바에 태그 위젯 마크업 추가
- `json_script`로 태그 데이터 주입

3. 스크립트
- `더보기`/`검색` 패널 토글
- 입력값 기반 자동완성(우선순위: 포함 위치, 빈도, 이름)
- 패널 외부 클릭 시 닫기

4. 스타일
- 데스크톱: 상위 태그 1줄 고정 + overflow-x
- 모바일: 상위 태그 줄 숨김, 버튼/패널 중심

## 검증
```bash
docker compose up -d
docker compose exec web python manage.py check
```
- 네비바에서 더보기/검색/추천 동작 확인
- 모바일 폭에서 레이아웃 깨짐 확인
